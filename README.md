# Mapbox by Buildup Bookings

A WordPress plugin that integrates Mapbox for displaying interactive maps with custom locations.

## Installation

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your Mapbox API token in the plugin settings

## Features

- Custom post types for managing Locations and Maps
- Interactive Mapbox map integration
- Advanced Custom Fields for location data management
- Shortcode support for embedding maps anywhere
- Admin interface for map configuration
- Filter locations by custom selections

## Usage

1. Add your Mapbox access token in the plugin settings
2. Create locations under **Locations** menu in WordPress admin
3. Create a new map under **Mapbox Maps** and optionally select specific locations
4. Copy the shortcode from the maps list (e.g., `[mapbox-map id="123"]`)
5. Paste the shortcode into any page or post to display your map
